## Projetos em destaque

### 1. HealthFlow Gov

**Stack:** Angular, TypeScript, SCSS e Vitest

Frontend que simula rotinas administrativas de uma plataforma municipal de gestão de saúde.

- Componentes standalone e carregamento de rotas com `loadComponent`.
- Organização por `core`, `shared`, `layout` e features de domínio.
- Serviços com dados mockados; a integração com sistemas legados é conceitual.

[Ver repositório](https://github.com/rodrygords/healthflow-gov)

### 2. API de Usuários em .NET

**Stack:** C#, ASP.NET Core, Entity Framework Core, SQLite e FluentValidation

API REST acadêmica para cadastro e gerenciamento de usuários com regras de validação e remoção lógica.

- Arquitetura em camadas com DTOs, serviços e repositórios.
- Validações de entrada e respostas de erro padronizadas com Problem Details.
- Hash de senha com PBKDF2 e testes automatizados; autenticação e autorização estão previstas como evolução futura.

[Ver repositório](https://github.com/rodrygords/API-de-usuario-as-Rodrygo)

### 3. ApiUsers com Spring Boot

**Stack:** Java 21, Spring Boot, Spring Data JPA, MySQL e Maven

API REST acadêmica para praticar cadastro e gerenciamento de usuários com persistência relacional.

- API REST desenvolvida com Spring Boot.
- Persistência com JPA/Hibernate, MySQL e identificadores UUID.
- Organização do código entre controller, service, repository e entity.

[Ver repositório](https://github.com/rodrygords/ApiUsers)

### 4. Sistema de Orçamentos para Telhados

**Stack:** C#, ASP.NET Core Web API, Entity Framework Core e SQLite

Backend para cadastrar clientes e administrar orçamentos de serviços de telhado.

- CRUD de clientes e orçamentos com DTOs e validações.
- Cálculo do valor total e controle do status dos orçamentos.
- Regra que impede excluir clientes vinculados a orçamentos.

[Ver repositório](https://github.com/rodrygords/sistema-orcamentos-telhados)

### 5. CRUD Angular

**Stack:** Angular, TypeScript, HTML, CSS e JSON Server

Aplicação frontend para cadastrar, consultar, editar e remover veículos.

- Formulários e componentes Angular para as operações de CRUD.
- Serviço responsável pelo acesso aos dados de veículos.
- JSON Server utilizado como API local simulada.

[Ver repositório](https://github.com/rodrygords/CRUD-ap2)

### 6. Aplicações Front-End

**Stack:** HTML, CSS, JavaScript e Fetch API

Atividade acadêmica estática para organizar uma lista de estudos e buscar sugestões de tarefas.

- Manipulação do DOM, eventos e validação de formulário com JavaScript.
- Consumo assíncrono da API pública JSONPlaceholder.
- Inclusão e remoção de itens com estado mantido em memória.

[Ver repositório](https://github.com/rodrygords/AP1-aplicacoes-front-end)
